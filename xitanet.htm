  Welcome To Xitami.Net
Xitami.Net is the domain dedicated to Xitami users on dynamic IP addresses. If you have a dial-up, xDSL, or cable-modem connection, your ISP gives you a fresh IP addresses each time you connect. This makes it hard to host a web server. Free dynamic-DNS services are not reliable enough. Our solution was to create Xitami.Net in cooperation with Tzolkin Corporation, one of the leaders in providing dynamic DNS.
How Xitami.Net Works
To use Xitami.Net, you choose a domain name, like 'myhost.xitami.net'. Each time you connect, the Xitami.Net software relays your new IP address to Tzolkin's servers, and the global DNS system. Anyone on the Internet can now find your server by its name, e.g. http://myhost.xitami.net/. This lets you create a real Internet web site on a dynamic IP address.

Acceptable Use Policy
Xitami.Net was created for the benefit of Xitami users and may not be used for these actions:
  • Running a mail server, and spamming others from a Xitami.Net E-mail Name from your Mail Server or another mail server using your Internet Name.
  • Using your Xitami.Net Host Name in a spam or UCE Email.
  • Hosting a "WAREZ" FTP or Web site that hosts copyrighted software for others to download for free.
  • Any other malicious act using your Xitami.Net DDNS Name.
