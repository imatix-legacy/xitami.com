Current Release 2.5c2
Package
Windows 9x/ME/XP/2000/2003
install package		 1.5Mb (xitami.com)
This package contains everything you need to run Xitami/2.5 under any 32-bit version of Windows. Includes a GUI version, a console version, and a service version and control panel (place this in the Windows System directory).
Unzip and run after reading the Xitami Documentation.
To upgrade an existing Xitami/2.5 installation, simply extract the executables into your current Xitami/2.5 directory. To migrate from 2.4 to 2.5, you must install 2.5 completely, then copy your configuration and security files and web pages.
Windows 9x/ME/XP/2000/2003
source package		 2.3Mb (xitami.com)
This package contains everything you need to build Xitami/2.5 from source under Windows. To build, unpack the source archive and run build.bat. Note: requires MSVC.
Linux, Unix Source Package 1.6Mb (xitami.com) This package contains everything you need to build Xitami/2.5 from source under Linux and Unix. To build, unpack the source archive and run build.sh.
Other operating systems Xitami/2.5 is portable to OpenVMS, OS/2, and other POSIX operating systems. The application may not have been ported and tested recently on such systems, and for production use we recommend a modern and well-patched operating system.
